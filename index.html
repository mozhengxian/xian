datetime Module

发表于 07-28-2018

datetime.datetime 类型

>>>datetime.datetime.now()
>>>datetime.datetime(2018, 7, 28, 7, 28, 17, 261975)

datetime.date 类型

>>>datetime.date.today()
>>>datetime.date(2018, 7, 28)

strftime(format)方法

将datetime.datetime类型转换成字符串

strptime(str, format)

将字符串转换为datetime.datetime格式

！注意：字符串的格式和转换的格式一致

>>>datetime.datetime.strptime(s, '%Y-%d-%m %H:%M:%S')
ValueError                            Traceback (most recent call last)
......
ValueError: time data '2018-07-28 07:44:22' does not match format '%Y-%d-%m %H:%M:%S'

最后返回的是datetime.datetime格式。

OS Module学习笔记

发表于 07-25-2018

os.getcwd( )

查询当前工作环境的路径。

os.getenv(key, defualt=None )

###os.environ.get(key, defualt=None)

获取系统环境变量，如果没有，返回None。

os.chdir(path)

改变当前工作环境的路径，可以是相对路径或绝对路径。

os.listdir(path=’.’)

默认值返回当前工作目录的文件，目录的列表。

os.walk(top, topdown=True, onerror=None, followlinks=False)

返回路径top目录下的子目录和文件列表。它的元素为元组.

以此类推。

如果topdown=False，则从子目录遍历起。

os.mkdir( path)

创建一个目录

os.makedirs( path)

创建多级目录

os.remove( )

删除文件

os.rmdir( )

删除目录

os.rename( )

重命名

os.path Module

os.path.basename(path)

返回该路径的文件名，如果没有，则返回None

os.path.dirname(path)

返回该目录的目录名

os.path.isdir(path)

判断路径是不是目录，不是返回False，是返回True

os.path.exists(path)

判断路径是否存在，返回True or False。

os.path.isfile(path)

判断路径是不是文件，返回True or False。

os.path.join(path, *path)

os.path.split(path)

返回一个元组，分别为path的目录路径，文件名。如果没有文件，返回空。

os.path.splitext(path)

返回path目录路径和文件名后缀。没有则返回空。

get（）方法：存入一个键，如果存在，则返回该键对应的值，不存在，则返回空。可以存入第二个指定的参数。

高阶函数：map，filter，reduce

map（）：接受两个参数，一个为函数，一个是可迭代的对象。功能：把这个函数作用于这个可迭代的对象的每一个元素。

filter（）：接受两个参数，第一个为自定义函数或None，第二个为可迭代的对象。功能：返回符合第一个函数的要求并为真的元素，如果是None，就返回为True的元素。

reduce（）：接受三个参数，第一个为自定义函数，第二个为可迭代对象，第三个为默认的基值。reduce（lambda x，y：x+y，[1，2，3，4]，10） 首先1，2 相加，然后相加的和3与3在相加。10作为开始时的基值。
