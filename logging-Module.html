logging Module

发表于 07-28-2018

Logger

logger 对象输出的错误级别：

NOTSET ：0

DEBUG：10

INFO：20

WARNING：30

ERROR：40

CRITICAL：50

输出到终端：

import logging
logger = logging.getLogger()  # 获取系统的Logger对象
logger

当前的Logger为root，默认级别为WARNING.

type(logger)


logger.warning('is when the event was loged')

默认输出格式：

'%(levelname)s:%(name)s:%(message)s'

设置INFO级别：

logger.setLevel('INFO')

低级别的不会输出

logger.info('is when this event was loged')
logger.debug('is when this event was loged')

logging.basicConfig()

输入到文件

logging.basicConfig(
    filename=filename.log,
    level=logging.debug,
    format='[%(levelname)s]:%(asctime)-10s :%(message)s,
    datefmt='%Y-%m-%d:%H:%M:%S'
    )

myapp.py 

import logging

def main():
    basicConfig(
        filename=myapp.log,
        level=logging.DEBUG,    # DEBUG要大写，or  level='DEBUG'
        format='[%(levelname)s]:%(asctime)-10s:%(message)s',
        datefmt='%Y-%m-%d:%H:%M:%S',
        )
    logging.debug('Starting....')
    logging.debug('Doing Somethings...')
    logging.debug('Finishing...')
    logging.info('starting...')
if name == '__main__':
    main()

setLevel()

Handler

FileHandler
StreamHandler

最佳使用logging的方案

import logging

logger = logging.getLogger()
handler = logging.StreamHandler()
formatter = logging.Formatter(
    '%(asctime)s %(name)-12s %(levelname)-8s %(message)s'
    ）
handler.setFormatter(formatter)
logger.addHandler(handler)
logger.setLevel(logging.DEBUG)

logger.debug('This is a %s', 'test')

内置日志格式

%(name)s        
生成日志的Logger名称

%(levelno)s        
数字形式的日志级别，包括DEBUG,INFO,WARNING,ERROR和CRITICAL。

%(levelname)s        
文本形式的日志级别，包括DEBUG,INFO,WARNING,ERROR和CRITICAL。

%(pathname)s        
输出该日志的语句所在的原文件的完整路径（如果可用）
。
%(filename)s        
文件名。

%(module)s        
输出该日志的语句所在的模块名。

%(funcName)s        
调用日志输出函数的函数名。

%(lineno)d        
调用日志输出函数的语句所在的代码行。

%(created)f        
日志被创建的时间，UNIX标准时间格式，表示从1970-1-1 00：00：00 UTC计算器的秒数。

%(relativeCreated)d        
日至被创建时间与日志模块被加载时间差，单位为毫秒。

%(asctime)s        
日志创建时间，默认格式是’2003-07-08 16:49:45,896’，逗号后为毫秒数。

%(msecs)d        
毫秒级别的日志创建时间

%(thread)d        
线程ID（如果可用）

%(threadName)s        
线程名称（如果可用）

%(process)d        
进程ID（如果可用）

%(message)s         
日志信息
